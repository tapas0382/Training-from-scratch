# 🚀 Day 7 – DOM Manipulation & Interactive Web Apps

## 📖 Overview

Day 7 of my **Time-Skip Software Engineering Journey** focuses on learning **DOM Manipulation** using JavaScript. The goal is to make web pages interactive by selecting HTML elements, modifying their content, changing styles, handling events, and building small real-world projects.

---

## 🎯 Objectives

* Learn how JavaScript interacts with HTML elements.
* Understand different DOM selection methods.
* Modify text, HTML, and CSS dynamically.
* Work with attributes and CSS classes.
* Build interactive web applications.

---

## 📚 Topics Covered

* DOM Introduction
* `getElementById()`
* `querySelector()`
* `querySelectorAll()`
* `innerText`
* `textContent`
* `innerHTML`
* Changing CSS using JavaScript
* Working with Attributes
* `classList.add()`
* `classList.remove()`
* `classList.toggle()`
* Event Handling
* Event Bubbling
* Event Delegation
