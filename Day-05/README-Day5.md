## 🧠 Key Concepts Learned

* Functions allow code reuse and improve readability.
* Parameters receive input, while `return` sends output back.
* Local variables exist only inside their function, whereas global variables are accessible throughout the program.
* Arrow functions provide a shorter and cleaner syntax for writing functions.
* Callback functions allow one function to be passed into another, enabling more flexible and asynchronous programming.
