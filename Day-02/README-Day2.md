# Day 2 - JavaScript Learning Journal

## 📌 Overview
On Day 2, I practiced writing reusable functions, working with objects, manipulating strings, and solving basic algorithmic problems in JavaScript. The goal was to improve problem-solving skills while understanding how functions interact with different data types such as numbers, objects, arrays, and strings.

---

## ✅ What I Learned

### 1. Building a Simple Calculator with Arrow Functions
I created a basic calculator by writing separate arrow functions for addition, subtraction, multiplication, and division.

Instead of writing one large function, I learned that breaking functionality into smaller reusable functions makes code cleaner, easier to test, and easier to maintain. This also reinforced the syntax and benefits of ES6 arrow functions.

---

### 2. Working with Objects and Arrays
I created a **student object** containing personal information and an array of marks. Then I wrote standalone functions that accepted the object as a parameter instead of attaching methods directly to the object.

Using these functions, I learned how to:

- Access object properties using dot notation.
- Print formatted information using template literals.
- Calculate the average of an array using `reduce()`.
- Add new values to an array using `push()`.
- Find the highest mark using `Math.max()` with the spread operator.

This exercise helped me understand how objects can store different types of data and how functions can operate on objects without modifying their overall structure.

---

### 3. Reversing a String Without Using `.reverse()`
I implemented a string reversal algorithm without relying on JavaScript's built-in `.reverse()` method.

By looping through the string from the last character to the first and building a new string, I learned how strings can be processed character by character. This strengthened my understanding of loops, indexing, and algorithmic thinking.

---

### 4. Counting Vowels in a String
I created a function to count the number of vowels in a string.

I stored all vowels inside a string and used the `includes()` method to check whether each character was a vowel while iterating through the input string.

This exercise improved my understanding of:

- String traversal
- Conditional statements
- Character matching
- Using built-in string methods effectively

## 💡 Takeaways

Day 2 showed me that JavaScript functions become much more powerful when they work with different data types like numbers, objects, arrays, and strings. I learned the importance of writing small, reusable functions instead of placing all logic in one block of code.
